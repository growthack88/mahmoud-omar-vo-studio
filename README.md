## نبذة

هذا الاستوديو يتيح لك توليد تعليقات صوتية عربية احترافية عبر Gemini AI بعدة لهجات (مصرية، سعودية، شامية، عراقية، جزائرية، مغربية، تونسية، خليجية، سودانية، يمنية، لبنانية، فصحى)، مع تحكّم كامل في حرارة الصوت، الانفعال، السرعة، العمق، طبقة الصوت، وشدّة التأثير الدرامي.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key.
3. Run the app:
   ```bash
   npm run dev
   ```

## Stack

- React 19 + Vite 6 + TypeScript
- `@google/genai` (Gemini 2.5 Pro / Flash / TTS)
- Tailwind (CDN) + custom CSS theme
